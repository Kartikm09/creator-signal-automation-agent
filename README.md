# Creator Signal Automation Agent

Automation project for collecting, scoring, and routing creator/content signals into a brief-ready workflow.

This repo is designed to support an AI automation / n8n / agentic workflow portfolio. It is public-safe and uses synthetic sample data.

## What It Does

- Ingests creator/content signals from a CSV export.
- Scores each signal using momentum, relevance, saturation, and effort.
- Generates a ranked brief queue.
- Flags signals that need human review.
- Includes an n8n-style workflow blueprint for automation handoff.

## Example

```bash
python -m creator_signal_agent.analyze examples/signals.csv
```
